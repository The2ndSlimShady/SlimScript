# If-Else Blocks

If-else blocks in SlimScript are just like the Lua's. One sample program should be enough for you to understand.

```
define age as 18

if >= age 18 then
    write "You're too old"
elif >= age 15 then
    write "You're a teen"
else
    write "Baby"
end

-- Output
--      You're too old
```
